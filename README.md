# Regulad Tap

## How do I install these formulae?

`brew install regulad/tap/<formula>`

Or `brew tap regulad/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "regulad/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
